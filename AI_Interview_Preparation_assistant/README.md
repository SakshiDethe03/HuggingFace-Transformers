# AI Interview Preparation Chatbot

An AI-powered Interview Preparation Chatbot built using Hugging Face Transformers and Gradio.  
This chatbot helps students prepare for technical interviews by providing simple and clear explanations of technical concepts.

## 🚀 Live Demo

🔗 [Hugging Face Space Link](https://huggingface.co/spaces/sakshidethe03/AI_Interview_Preparation_ChatBot)

## 📌 Features

- AI-powered interview preparation
- Simple and easy-to-understand explanations
- Interactive chatbot interface
- Real-time response generation
- Built using open-source LLMs
- Deployed on Hugging Face Spaces

## 🛠️ Technologies Used

- Python
- Hugging Face Transformers
- Gradio
- Qwen Models
- Google Colab

## ⚙️ Installation

Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/ai-interview-chatbot.git
```

Move into the project folder:
```bash
cd ai-interview-chatbot
```

Install dependencies:
```bash
pip install -r requirements.txt
```

Run the application:
```bash
python app.py
```

## 🧠 Model Used
Qwen2.5-1.5B-Instruct

## 💡 How It Works
1. User enters a technical interview question
2. The prompt is sent to the LLM
3. The model generates a response
4. Gradio displays the response in chatbot format
